# NeuroLight Project Documentation

Welcome to the NeuroLight project documentation hub. This site contains all meeting notes, design documents, and project documentation for the NeuroLight capstone team.

## About NeuroLight

NeuroLight is a Python desktop application designed to automate neuronal activity analysis in circadian rhythm research. The application provides researchers with tools for image alignment, ROI selection, and time-series analysis of calcium imaging data.
